About

Here is some information about Shooty Game and its developers.

Shooty Game is a simple shooter game made with p5.js!

In 2020, Kang Yiming created the game with p5.js for our school's Computer Studies project. Then, the game gained popularity from many students of the school, and he continued to develop and improve the game.

In 2021, Cen Han built a website for the game, so that Shooty Game can possibly expand beyond the school context. Notably, the website is also built for our school's Computer Elective Programme project.
